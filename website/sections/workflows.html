Workflows

Design auditable, end-to-end security operations

SAPL workflows codify organisational knowledge. Blend automation with human approvals to deliver consistent playbooks across detection, response, and proactive testing.

Prototype rapidly

Iterate in sapl shell with live data. Use watch mode to reload plugins when underlying Python modules change.

Validate collaboratively

Run sapl inspect --json to produce machine readable manifests. Teams review diffs inline and sign off on guardrails before production rollout.

Operationalise safely

Deploy workflows with scheduled runs or event driven triggers. Every invocation is logged, immutable, and exportable to SIEM platforms.

Documentation as a service

Publish the advanced website to give stakeholders a live view into workflows, controls, and escalation paths. QR codes embedded in runbooks link to the exact section on this site, streamlining audits and post-incident reviews.
